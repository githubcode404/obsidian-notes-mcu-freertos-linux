## 3-1
##  开启**任务调度器**：

**位置**：main -> osKernelStart( )
其中 **vTaskStartScheduler( )** 是freertos自带函数：启动任务调度器
```cpp
osStatus_t osKernelStart (void) {
  osStatus_t stat;

  if (IS_IRQ()) {
    stat = osErrorISR;
  }
  else {
    if (KernelState == osKernelReady) {
      KernelState = osKernelRunning;
      //这一行是核心
      vTaskStartScheduler();
      //
      stat = osOK;
    } else {
      stat = osError;
    }
  }

  return (stat);
}
```

## 3-2-1


**ARM架构**
- 单片机（SOC system on chip）：CPU+ flash（存程序）+ 内存（数据）
- CPU：计算单元+寄存器（存数据）

**执行程序**：a = a * b;
1. CPU读取flash，得到代码命令（汇编语言）
2. CPU读取内存，得到a，b的值，存入寄存器
3. CPU计算单元计算a * b，存入寄存器
4. CPU向内存a的地址写寄存器的值

## 3-2-2

TIPS:
- 栈位于内存的高地址位置，方向是从高地址向低地址
- 堆也位于内存中，位于低地址（相较于栈来说），方向是从低地址向高地址
- push 入栈，参数从右往左读
