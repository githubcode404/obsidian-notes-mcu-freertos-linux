## 开启**任务调度器**：

**位置**：main -> osKernelStart( )
其中 **vTaskStartScheduler( )** 是freertos自带函数：启动任务调度器
```cpp
osStatus_t osKernelStart (void) {
  osStatus_t stat;

  if (IS_IRQ()) {
    stat = osErrorISR;
  }
  else {
    if (KernelState == osKernelReady) {
      KernelState = osKernelRunning;
      //这一行是核心
      vTaskStartScheduler();
      //
      stat = osOK;
    } else {
      stat = osError;
    }
  }

  return (stat);
}
```

BaseType_t xTaskCreate( TaskFunction_t pxTaskCode, // 函数指针, 任务函数
