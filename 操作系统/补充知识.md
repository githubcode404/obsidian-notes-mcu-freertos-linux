|             | Socket 文件             | Ext4 文件             |
| ----------- | --------------------- | ------------------- |
| **存在位置**    | 内存（或文件系统仅作命名）         | 磁盘块                 |
| **数据持久性**   | 易失性（进程结束即消失）          | 持久性                 |
| **读写对象**    | 另一个进程/网络端点            | 磁盘块                 |
| **内核结构**    | `struct socket` → 协议栈 | `struct inode` → 块层 |
| **文件大小**    | 无固定大小（流式）             | 明确的字节数              |
| **seek 操作** | 不支持（不可随机访问）           | 支持 `lseek`          |

|      | Section                  | Segment                  |
| ---- | ------------------------ | ------------------------ |
| 位置   | 磁盘文件内                    | 内存运行时                    |
| 视角   | 链接器/静态                   | 操作系统/动态                  |
| 粒度   | 更细（只读数据、代码等分开）           | 更粗（权限相同的 section 合并）     |
| 数量关系 | 一个 segment 可包含多个 section | 一个 section 只属于一个 segment |
