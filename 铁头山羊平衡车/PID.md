**理论过程**：
对被控对象建立数学模型，拉普拉斯变换得到输入输出的传递函数，测量实际数据带入函数，根据传递函数绘制波特图，对标理想波特图得到所需pid波特图，计算得到PID的系数


**控制系统整体**![[Pasted image 20260503152654.png]]

**波特图**：
- 幅值：无单位（因此pid函数的波特图和传递函数的波特图可以直接叠加
- 角频率：单位为rad/s



**传递函数**：
- 从电机电压得到电机输出角速度
- 本身**固定**，和硬件有关
- 找到传递函数，才知道pid系数如何设定
**量纲**：rad/s  /  v![[Pasted image 20260503152549.png]]
**PID系数**：从角速度差得到电压![[Pasted image 20260503153715.png]]



periodic 宏定义实现：间隔固定时间，执行函数


**电机调速代码**：
- 初始化左右：电机+编码器+pid参数
- 设定目标角速度（左右电机分开，涉及转弯时速度不一致
- while循环**包括**：
-  宏定义控制时间间隔
- 得到当前速度
- 计算输出电压
- 计算占空比
- 设定占空比



**改进：**
- 输出限幅：输出电压上下限由电池决定，超过上下限无意义（初始化时明确上下限）
- 积分限幅：积分项过大，导致kp失效
- pid复位：电机关闭后，清除暂存数据（standby控制电机是否启动



**串级PID**

内层：车轮胎前后运动的角速度
中层：车身倾斜角速度变化率（陀螺仪）
外层：车身倾斜角度（加速度计）


车身倾斜角度 -> 角度环 -> 目标车身倾斜角速度 -> 车身角速度环 -> 目标车轮角速度 -> 车速环 
-> 车身实际速度
